Quick Tip

A security tip this time, important for anyone who searches for code to use on the command line (all of us?!). Graduate student Suresh Alse has an elegant but scary demo showing how risky copying and pasting commands is. Sure, we already know to fully understand what commands to before pressing Enter, but there's another risk — hidden code.

His demo uses CSS to hide code within a simple ls -lat command. It looks innocent but demonstrates that when you select and copy that code from his website, any text hidden with CSS will also be copied and consequently executed. Fortunately his example is harmless but scary nevertheless. Definitely worth trying out:

lifepluslinux.blogspot.jp/2017/01/look-before-you-paste-from-website-to.html

So how do we protect ourselves from such an attack? There are a couple of options:

  • Look at the command and then type it out the command manually
  • Copy the command and paste it into a text editor to double-check the contents

It may seem like extra effort but worth it to learn more about the commands you're executing.
